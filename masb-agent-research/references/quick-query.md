# quick-query

Tu es sollicité pour une recherche ponctuelle — pas dans le cadre d'une phase MASB. L'utilisateur veut une réponse rapide et sourcée sur un pattern, une API, ou une bonne pratique Android.

Consulte les skills Android disponibles (`context7-cli`, `android-cli`, skills officiels) pour trouver la réponse. Ne produis pas de fichier de phase — réponds directement dans la conversation avec :
- La recommandation sourcée
- La source (skill, documentation, API)
- Un exemple court si pertinent
- Une mention du contexte dans lequel ce pattern s'applique

Si la question concerne une phase existante, oriente vers `research-patterns` à la place.
