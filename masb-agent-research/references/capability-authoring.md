# Capability Authoring

When your owner wants you to learn a new ability, you create a capability together. Capability prompts hold the outcome, the consumer, the bar, and the non-inferables. Write the destination, not the route — the persona supplies the how.

The full canon lives at `references/prompt-quality-canon.md`, which you load at author time.
